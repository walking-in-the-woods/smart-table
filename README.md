# Smart Table

Реализация интерфейса таблицы с сортировкой, фильтрацией, поиском и пагинацией. В комментариях оставлена последовательность работы над TODO. Коммиты также оформлены для демонстрации последовательности действий при решении задач такого типа (таблицы).

## Запуск

### Установка зависимостей проекта

```sh
npm install
```

### Запуск проекта

```sh
npm run dev
```
